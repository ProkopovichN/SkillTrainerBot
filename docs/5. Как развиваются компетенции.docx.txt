Как развиваются компетенции

Быстрый диагностический фильтр перед “Конкретными действиями”:
Знания: “не знаю как” → обучаем/даём материалы/разборы
Навык: “знаю, но не получается стабильно” → практика, обратная связь, повторение
Убеждения/установки: “не верю / избегаю / боюсь” → работа с установками, безопасные эксперименты, поддержка	

В Авито, планируя развитие, используем подход 70-20-10
70% обучения происходит в процессе работы (через опыт, задачи, вызовы),
20% — через взаимодействие с другими (обратная связь, наставничество, коучинг),
10% — через формальное обучение (курсы, тренинги, лекции).

Подробнее по компонентам:

70% — Обучение через практику. Это основной двигатель роста. Сотрудник развивается, когда:
* берёт на себя новые или более сложные задачи,
* работает над реальными бизнес-проблемами,
* получает ответственность за результат,
* проходит через неудачи и рефлексию.
Пример для аналитика: вместо курса по «стратегическому мышлению» — участие в квартальной стратег-сессии с продактами, где нужно предложить гипотезы роста на основе данных.

20% — Обучение через других. Развитие происходит в диалоге и взаимодействии:
* регулярная обратная связь от руководителя и коллег,
* менторинг или коучинг,
* совместное решение задач,
* участие в комьюнити практиков.
Пример: после каждого демо аналитик запрашивает фидбэк не только по данным, но и по тому, насколько ясно он донёс выводы разным аудиториям.

10% — Формальное обучение (Formal Learning). Это поддержка, а не основа:
* корпоративные тренинги,
* онлайн-курсы,
* чтение книг,
* конференции.
Важно: формальное обучение работает только тогда, когда оно интегрировано в практику (70%) и обсуждается с другими (20%). Иначе знания быстро забываются.

Как применять 70-20-10 при составлении ИПР:
Для каждой цели развития задавайте:
70%: Какую рабочую задачу/проект можно использовать для развития?
20%: Кто может дать фидбэк, поддержку, коучинг?
10%: Какой минимальный формальный ресурс поможет (если вообще нужен)?
